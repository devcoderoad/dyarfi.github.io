What I Do

Hello my name is Defrian and I work in digital agency in Jakarta, Indonesia. Most of the time my job was developing web pages, handling servers and exploring new technique on the web. I've been in this industries for quite sometimes and there's a lot of things to keep up with our standard line of work. Internet are always evolving themself into something more than they are right now. Websites are becoming less boring than they were first discovered in the 90s. Today, more people used internet for their daily bassist and more devices to support them to connect to internet. In my line, web development is what I do to support them.

Specialized

Web programming with PHP frameworks such as : Laravel, CakePHP, Yii, CodeIgniter or native PHP, also found joy slicing PSD to HTML for Frontend Development. CMS such as Wordpress, OpenCart and Drupal has been also part of my working environment. UI Development for frontend and backend with HTML, CSS and Javascript / Ajax also ActionScript 2/3 in the past.

Familiar with : Laravel, Yii, CakePHP, CodeIgniter, Kohana, jQuery, jQuery UI, Bootstrap, Gulp, Webpack, React, Jekyll, HTML 5, CSS / SASS, Javascript & Ajax and RESTful API.
Development Tools : NetBeans 7.0 / Visual Studio Code / Sublime Text, XAMPP / Laragon / Homestead / Docker / Node, Adobe CS3, Git & SVN, Terminal, Shell, Win 7 and Mac OSX.

Related day to day knowledge:

  • HTML, CSS / SCSS, Javascript and AMP Pages
  • PHP & MySQL, PostgreSQL, SQLite & MongoDB / Firebase (NoSQL)
  • Slicing PSD to HTML Template / Webpack / Gulp & Jekyll
  • HTML Email Templates (Marketing Automation / Mailchimp)
  • NodeJS & NPM
  • Arduino and ESP8266 (Internet of Things) / C++
